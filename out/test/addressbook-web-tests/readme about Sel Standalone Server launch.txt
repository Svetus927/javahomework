узнаем ip адресс машины, на которой хотим запусать браузер командой: ipconfig /all

создаем файл св-в, например remote.properties,  в котором прописать все настройки нужной машины

в AppManager.init при инициализации драйвера дб добавлена проверка на св-во selenium.server: если пусто то
инициализируется WebDriver, если не пусто то RemoteWebDriver
